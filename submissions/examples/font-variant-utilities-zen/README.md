# Font Variant Utilities

CSS utility classes for font variant settings.

## Classes

- `.font-normal` — `font-variant: normal`
- `.font-small-caps` — `font-variant: small-caps`
- `.font-all-small-caps` — `font-variant: all-small-caps`

## Why

Quickly apply small-caps typography for headings, acronyms, or stylistic text without custom CSS.
